怎样将域名绑定到github pages 博客上

2018-10-12
github

1、添加CNAME文件

在仓库根目录下添加一个CNAM文件，没有后缀名，里面内容为你的域名(如:test.com),不需要添加http/www等前缀。

ping yougithubname.github.io记录下IP地址

2、添加 DNS Service记录

去DNSPod注册个账号，添加域名，设置两个A记录。分别是@和w w w，ip地址填上个步骤获取的IP地址。

3、设置域名的DNS

在相应域名的Csutom DNS里，设置DNS service,添加两条记录f1g1ns1.dnspod.net和f1g1ns2.dnspod.net

4、漫长的等待

要全球解析生效，得等上一会了，也可以先ping一下自己的设置对不对。
